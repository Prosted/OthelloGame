## protocol

* 100 좌표를 이용해 플레이 함 request.
* 101 서버로부터 메세지를 받음 response.
* 102 좌표를 이용해 플레이함 response.
* 104 채팅 request.

* ~~200 방을 만듬 request.~~
* ~~201 방이 만들어짐 response.~~
* 202 방에 입장 request.
* 203 접속 종료 request.
* 204 방에 입장 response.
* 205 접속 종료 response. 

* 301 히스토리 요청 request.
* 302 히스토리 응답 response.

* 400 게임 시작 요청 request.
* 401 게임 시작 응답 response.
* 402 게임 끝 요청 request.
* 403 게임 끝 응답 response.
